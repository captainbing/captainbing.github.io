配置环境变量

目的：随时能够使用Java工具相关的一些命令

操作：将jdk环境下的bin目录的路径粘贴到环境变量中

JDK和JRE

jdk包含jre

jdk: java development kit

jdk包含 JVM 核心类库 开发工具

jre: java runtime environment

java的运行环境，可以运行字节码文件

注释

变量

  • 变量定义格式

数据类型 变量 = 数据值

  • 基本数据类型

byte 字节型 1字节

short 短整型 2字节

int 整型 4字节

long 长整型 8字节 需要加入L标识

float 单精度 4字节 需要加入F标识

double 双精度 8字节

boolean true false

char 字符型 2字节

  • 引用数据类型

数据值是存在其他空间中的，自己空间中存储的是地址值

String 字符串

Array 数组

对象

标识符

由数字，字母，下划线，$符号组成

不能是数字开头，不能是关键字，区分大小写

变量，方法名 小驼峰命名法 firstName

类名 大驼峰命名法 HelloWorld

运算符

算数运算符

符号 作用
+ 加
- 减
* 乘
/ 除
% 取模，求余

隐式转换

范围：byte < short < int < long < float < double

取值范围小的数 ===> 取值范围大的数

byte，short ，char做运算的时候会被提升到int类型

强制转换

取值范围大的数 ===> 取值范围小的数

格式：目标数据类型 变量名 = (目标数据类型) 被强转的数据

自增自减运算符

赋值运算符

符号 作用
= 赋值
+= 加后赋值
-= 减后赋值
*= 乘后赋值
/= 除后赋值
%= 取余后赋值

关系运算符

== 等于 != 不等于 > 大于 >= 大于等于 < 小于 <= 小于等于

==

基本数据类型中比较值

引用数据类型比较地址

逻辑运算符

符号 作用 说明
& 与 左右为真都为真
| 或 有一个为真，则为真
！ 非 取反
^ 异或 相同为假，不同为真
&& 短路与 已经可以判断结果，后面不执行
|| 短路或 已经可以判断结果，后面不执行

三元运算符

关系表达式 ? 表达式1 : 表达式2

关系表达式为true，执行表达式1

关系表达式为false，执行表达式2

流程控制语句

循环控制语句

continue与break

continue 跳出本次循环，继续下次循环

break 结束本次循环

数组

完整声明：数据类型[] 数组名 = new 数据类型[]{元素1,元素2...}

简化格式：数据类型[] 数组名 = {元素1,元素2...}

内存地址格式：[I@1b6d3586

[ 表示当前是一个数组

I 表示当前数组的类型是int型

@ 间隔符号(固定格式)

1b6d3586 地址值(十六进制)

数组长度属性：length

idea快速生成数组的遍历形式： 数组名.fori

数组内存图

方法

基本概念

方法是程序中最小的执行单元

提高代码的复用性和可维护性

形参：方法中定义的参数

实参：方法调用中的参数

权限修饰符：

修饰符 同一个类中 同一个包中其他类 不同包下的子类 不同包下的无关类
private ✔
友好 ✔ ✔
protected ✔ ✔ ✔
public ✔ ✔ ✔ ✔

方法重载

同一个类中，方法名相同，参数不同的方法，与返回值无关

参数不同：个数不同、类型不同、顺序不同

面向对象

类(class)： 共同特征的描述

对象(new)： 真实存在的具体实例

三大特征： 封装、继承、多态

变量的类型

成员变量

类中声明的变量

静态变量

类中使用static修饰的变量，属于类，不属于对象

静态方法中只能访问静态变量和静态方法

非静态方法可以访问所有

静态方法中无this关键字

局部变量

方法中声明的变量

JavaBean

this

本质：所在方法调用者的地址

可以区别成员变量和局部变量

类中每个方法都有一个默认参数this

super

代表父类存储空间

访问父类的构造方法，成员变量，成员方法

new

new出来的对象都在堆里开辟空间

String name = "小明" 小明存在于字符串常量池中

String name = new String("小红") 小红存在于堆内存中

final

final修饰基本类型，数据值不能变 常量

final修饰引用类型，数据地址不能变

StringBuilder

提高字符串的操作效率

StringJoiner

对象内存图

继承

格式

public class 子类 extends 父类{}

子类可以继承父类中的属性和行为

子类可以在父类的基础上增加其他功能

只能单继承，不能多继承，一个类只能继承一个直接父类

所有的类都直接或间接继承Object类

重写

父类中的方法满足不了需求，，需要进行方法重写

重写方法的名称和形参必须与父类中的一样

访问权限子类必须大于等于父类

返回值类型子类必须小于等于父类

多态

前提

有继承/实现关系

有方法重写

Animal a = new Dog();

调用成员变量特点：编译看左边，运行看左边

调用成员方法特点：编译看左边，运行看右边

包

包就是文件夹，管理不同功能的java类

书写规则：公司域名反写 + 包的作用，见名知意

全类名：包名 + 类名

代码块

局部代码块：在方法内部声明,生命周期短 { }

构造代码块：在类中声明，创建一次对象执行一次，构造方法之前 {}

静态代码块：加载类的时候执行，只执行一次 static{}

抽象类

关键字：abstract

抽象类不能实例化，但可以有构造方法

具体实现的方法和抽象方法可以同时存在

子类抽象方法必须实现，或者子类也是抽象方法

接口

关键字：interface,implements

接口不能实例化，和类之间是实现关系

接口中的方法子类必须全部重写，或者子类是抽象类

实现类可以同时实现多个接口

成员变量默认使用public static final 修饰 只能是常量

成员方法默认使用public修饰，子类实现方法中访问权限不能低于public

JDK8新特性

接口中可以定义有方法体的方法

JDK9新特性

接口中可以定义私有方法

内部类

成员内部类

应用方面：汽车的发动机，ArrayList中的迭代器

内部类可以直接访问外部类，外部类需要实例化内部类才能访问

在其他类中访问成员内部类：

Outer.Inner in = new Outer().new Inner();

静态内部类

只能访问外部类的静态变量和静态方法，访问非静态需要创建对象

外部类名.内部类名 对象名 = new 外部类名.内部类名()

局部内部类

定义在方法内部

外界无法直接使用局部内部类，需要在方法中创建对象

可以直接访问外部类的成员和该方法的局部变量

匿名内部类

隐藏了名字的内部类，底层同时实现了创建子类和对象的过程

new 类/接口(){

​ 重写方法;

}

包含了继承或实现，方法重写，创建对象

整体就是一个类的子类对象或者接口的实现类对象
